# Boilerplate

### Setup
 1. Initialize virtual env: `make venv`
 2. Activate it with: `. env/bin/activate`
 3. Install dependencies: `make install`

### Notes
- Run application: `make run`
- Run unit tests (whole codebase): `make test`
- Make code pretty: `make format`
- Lint code: `make lint`
